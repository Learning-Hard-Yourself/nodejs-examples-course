# TCP Chat

Real-time chat using TCP sockets with the net module.

## Features

- ✅ TCP server with multiple clients
- ✅ Message broadcasting
- ✅ Client connection management

## Usage

Start server:
```bash
node server.js
```

Connect clients:
```bash
telnet localhost 8080
```

Or use client:
```bash
node client.js
```

## Learn

- TCP sockets with net module
- Event-driven networking
- Broadcasting to multiple clients
