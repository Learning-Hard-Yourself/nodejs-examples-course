# Stream Processor

Process large files efficiently using Node.js streams.

## Features

- ✅ Transform streams for data processing
- ✅ Pipeline for error handling
- ✅ Memory-efficient large file processing
- ✅ Custom transform operations

## Usage

```bash
node processor.js input.txt output.txt
```

## Learn

- Streams and backpressure
- Transform streams
- Pipeline for error handling
